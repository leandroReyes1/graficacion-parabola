## 📖 Cómo Usar el Graficador

1.  Abre el archivo `index.html` en tu navegador (se recomienda usar un servidor local como **Live Server** o **XAMPP**).
2.  En el campo de texto **f(x) =**, escribe la función que deseas graficar.
    * *Nota*: Recuerda usar el asterisco `*` para multiplicaciones (ej: `2 * x + 1`).
3.  Haz clic en el botón **"Graficar"**.
4.  El plano se ajustará automáticamente para mostrar la curva en color rojo.
